![CMake](https://img.shields.io/badge/CMake-3.12+-green.svg)
![Doxygen](https://img.shields.io/badge/Documentation-Doxygen-brightgreen.svg)
![Windows](https://img.shields.io/badge/Platform-Windows%2010+-lightgrey.svg)

Генератор штриховки для SLM-слайсера - программа для создания линий штриховки внутри прямоугольных контуров с настраиваемым углом наклона и шагом.

## Возможности

- **Генерация штриховки** под произвольным углом (0-180°)
- **Поддержка прямоугольных контуров** с автоматической валидацией
- **Двойной вывод результатов**: текстовый формат и SVG визуализация
- **Гибкие параметры**: угол наклона, шаг между линиями
- **Полная документация** с Doxygen

## Установка и сборка

### Предварительные требования

- **Windows 10** или выше
- **CMake** 3.12 или выше
- **Компилятор C++20** (MSVC, MinGW-w64)
- **Doxygen** (опционально, для документации)

## Сборка проекта

```bash
git clone https://github.com/pythshom2228/hatch_generator.git
cd hatch_generator
mkdir build
cd build
```

### MinGW
```bash
cmake .. -G"MinGW Makefiles"
cmake --build . 
```
### MSVC
Открыть проект в Visual Studio

## Генерация Документации
```bash
doxygen Doxyfile
start .\doc\html\index.html
```


### Командная строка

```bash
# Базовое использование
hatch_generator.exe --angle 45 --step 1

# Пример с другими параметрами
hatch_generator.exe --angle 30 --step 0.5
hatch_generator.exe --angle 90 --step 2
```

### Интерактивный режим

После запуска программа запросит:

1. **Координаты прямоугольника** - 4 точки по порядку (по или против часовой стрелки)
2. **Параметры штриховки** (опционально) - угол и шаг, либо используются значения по умолчанию

Пример ввода координат:
```
Введите x y для 1 точки: 0 0
Введите x y для 2 точки: 10 0  
Введите x y для 3 точки: 10 10
Введите x y для 4 точки: 0 10
```

## Форматы вывода

### Текстовый вывод (`output.txt`)

```
Line 1: (0,0) -> (10,10)
Line 2: (0,1) -> (9,10)
Line 3: (0,2) -> (8,10)
...
```

### Визуализация (`hatch_rectangle.svg`)

SVG файл с:
- Контуром прямоугольника (черная обводка)
- Линиями штриховки (черные линии)
- Масштабированием для лучшего отображения

### Пример 1: Квадрат с диагональной штриховкой

```bash
hatch_generator.exe --angle 45 --step 1
```

Ввод координат:
```
(0,0), (10,0), (10,10), (0,10)
```

### Пример 2: Прямоугольник с вертикальной штриховкой

```bash
hatch_generator.exe --angle 90 --step 0.5  
```

Ввод координат:
```
(0,0), (15,0), (15,10), (0,10)
```
